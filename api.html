Instant Todo List API

GET /
- Return this page

GET /todos
- Return an array of all the todos [{ id: "", name: "", done: false }, {...}, ...] 

POST /todo
Create a Todo
- Body { name: "", done: true } or { name: "" }
- Return the created Todo { id: "", name: "", done: false }

PUT /todo/[id]
Update a Todo
- Body { name: "", done: false } (both properties are optional)
- Return the updated todo

DELETE /todo/[id]
Delete a Todo
- Return an empty response
